# simple-webserver2 : ファイルの内容を返すWebアプリケーション

**書籍での説明箇所** :「6.2.2 ファイルの内容を返すアプリケーション」p157

## 概要

あらかじめ用意されたファイルの内容を返すWebアプリケーションです。

`http://localhost:8080/` へアクセスすると、 `static/hello.html` の内容を返します。
